# **App Name**: WooNext Shop

## Core Features:

- Product Listing: Display a list of products fetched from the WooCommerce REST API.
- Product Details: Show detailed information for a selected product.
- Shopping Cart: Enable users to add products to a shopping cart and view cart contents. Cart state is maintained in local storage.
- AI-Powered Product Recommendations: Use AI to recommend products related to those in the cart, enticing the user to add more.  The AI tool decides when enough products are in the cart to make related recommendations relevant and useful.
- Checkout Redirection: Redirect users to the WooCommerce checkout page to complete their purchase.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern and sophisticated feel.
- Background color: Light grey (#F5F5F5), providing a clean backdrop to make the products stand out. 
- Accent color: Electric Indigo (#6F00FF), drawing attention to important interactive elements and AI suggestions.
- Body and headline font: 'Inter', a sans-serif for a clean and modern look, providing excellent readability on screens.
- Use simple, minimalist icons to represent different product categories and actions.
- Use a grid-based layout to display products, ensuring a clean and organized presentation.
