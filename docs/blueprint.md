# **App Name**: Blogtastic

## Core Features:

- Blog Post Display: Display blog posts with titles, excerpts, and publication dates.
- Category Filtering: Allow users to filter blog posts by category.
- Tag Filtering: Enable users to filter blog posts by tags.
- Search Functionality: Implement a search bar to find blog posts by keywords. LLM can act as a tool to locate keywords in each blog.
- Author Information: Show author name and photo for each blog post.
- Archive View: Provide an archive page to view older posts by month and year.

## Style Guidelines:

- Primary color: Soft Blue (#A0C4FF) for a calming and trustworthy feel.
- Background color: Light Gray (#F0F0F0) to ensure readability and a clean design.
- Accent color: Muted Purple (#BDB2FF) to highlight key elements like links and buttons.
- Body and headline font: 'Alegreya', serif, for an elegant, intellectual, contemporary feel.
- Simple, outlined icons to represent categories and tags.
- Clean and modern layout with ample white space for easy readability.
- Subtle transitions on hover for interactive elements.
